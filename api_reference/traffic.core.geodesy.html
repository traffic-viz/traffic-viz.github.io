traffic.core.geodesy

This module contains a set of geodesy functions wrapped around all what is available in the pyproj library. All angles are in degrees, all distances are in meter.

traffic.core.geodesy.bearing(lat1, lon1, lat2, lon2, *args, **kwargs)

Computes the distance(s) between two points (or arrays of points).

Parameters:

  • lat1 (TypeVar(F, float, Iterable[float], List[float])) – latitude value(s)

  • lon1 (TypeVar(F, float, Iterable[float], List[float])) – longitude value(s)

  • lat2 (TypeVar(F, float, Iterable[float], List[float])) – latitude value(s)

  • lon2 (TypeVar(F, float, Iterable[float], List[float])) – longitude value(s)

Return type:

TypeVar(F, float, Iterable[float], List[float])

Returns:

the bearing angle, in degrees, from the first point to the second

traffic.core.geodesy.destination(lat, lon, bearing, distance, *args, **kwargs)

Computes the point you reach from a set of coordinates, moving in a given direction for a given distance.

Parameters:

  • lat (TypeVar(F, float, Iterable[float], List[float])) – latitude value(s)

  • lon (TypeVar(F, float, Iterable[float], List[float])) – longitude value(s)

  • bearing (TypeVar(F, float, Iterable[float], List[float])) – bearing value(s)

  • distance (TypeVar(F, float, Iterable[float], List[float])) – distance value(s)

Return type:

tuple[TypeVar(F, float, Iterable[float], List[float]), TypeVar(F, float, Iterable[float], List[float]), TypeVar(F, float, Iterable[float], List[float])]

Returns:

a tuple with latitude value(s), longitude value(s) and bearing from the destination point back to the origin, all in degrees.

traffic.core.geodesy.greatcircle(lat1, lon1, lat2, lon2, *args, **kwargs)

Computes a list of points making the great circle between two points.

Parameters:

  • lat1 (float) – latitude value

  • lon1 (float) – longitude value

  • lat2 (float) – latitude value

  • lon2 (float) – longitude value

Return type:

list[tuple[float, float]]

Returns:

a tuple with latitude values, longitude values, all in degrees.

traffic.core.geodesy.mrr_diagonal(geom)

Returns the length of the diagonal of the minimum rotated rectangle around a given shape.

Consider using a project_shape() method before applying this method if you need a distance in meters.

Return type:

float
