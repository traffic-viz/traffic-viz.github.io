traffic.core.aero

This module implements functions for aeronautics, conversions between:

  • altitude, temperature, pressure, density in the International Standard Atmosphere model

  • computed air speed (CAS), true air speed (TAS), equivalent air speed (EAS), Mach number

All physical quantities assume standard units (SI), angles are all in degrees.

traffic.core.aero.atmos(h)

International Standard Atmosphere calculator

Parameters:

h (float) – altitude in meters \(0 < h < 84852\) (will be clipped when outside range, integer input allowed)

Return type:

Tuple[float, float, float]

Returns:

  • the pressure (in Pa)

  • the air density \(\rho\) (in kg/m3)

  • the temperature (in K)
